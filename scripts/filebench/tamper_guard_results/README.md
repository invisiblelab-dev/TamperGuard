# Filebench results

This directory contains the results of the filebench experiments ran with filebench_repeats.sh script on 13/02/2026 in a Almalinux 9.6 server with 4 vCPUs and 16GB RAM.

Testing of filebench workloads with different configurations:
- File System
- Modular Lib with FUSE Local
- Modular Lib with Fuse Anti-Tampering Local
- Modular Lib with Fuse Anti-Tampering IPFS

Filebench workloads:
- webserver.f
- fileserver.f
- varmail.f
